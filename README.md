# Agentforce Revenue Management - Transaction Line Editor Toggle Component

This repository contains a Lightning Web Component (LWC) that automatically enables the Toggle Instant Pricing feature when loading the Transaction Line Editor in [Agentforce Revenue Cloud](https://www.salesforce.com/eu/sales/revenue-lifecycle-management).

## Overview

The component provides an automated solution for the [Transaction Line Editor](https://help.salesforce.com/s/articleView?id=ind.qocal_view_and_edit_quotes.htm&type=5), ensuring that Instant Pricing is enabled by default when users access the quote editing interface. This enhancement streamlines the revenue management workflow by eliminating the need for manual toggle activation.

## Features

- **Automatic Toggle Activation**: Instantly enables the Toggle Instant Pricing feature upon page load
- **Seamless Integration**: Works directly with the standard Transaction Line Editor interface
- **Enhanced User Experience**: Reduces manual steps in the quote management process
- **Revenue Cloud Compatible**: Fully integrated with Agentforce Revenue Management capabilities

## Use Case

This component is particularly useful for organizations that:
- Require instant pricing to be enabled by default for all quote transactions
- Want to standardize the user experience across their sales teams
- Need to reduce manual configuration steps in the revenue management process

## Installation
Deploy this Lightning Web Component to your Salesforce org with Agentforce Revenue Cloud and drop it on the Page before Transaction Line Editor component.

## Requirements
- Salesforce org with Agentforce Revenue Cloud license
- Access to Transaction Line Editor functionality
- Lightning Web Component deployment capabilities
